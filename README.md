#Attention: This repository has been abandoned.

We had separated huge moztw-central repo, into following isolated sub-domain-based repos,

* [base](https://github.com/moztw/base) - Server configurations and automatic scripts
* [www.moztw.org](https://github.com/moztw/www.moztw.org)  - Main MozTW website
* [forum.moztw.org](https://github.com/moztw/forum.moztw.org) - MozTW Forum
* [gfx.tw](https://github.com/moztw/gfx.tw) - Repo for http://gfx.tw, personal Firefox promotion and product page platform
* [irclog.moztw.org](https://github.com/moztw/irclog.moztw.org) - IRC Archive for #mozilla-taiwan of irc.mozilla.org
* [translate.moztw.org](https://github.com/moztw/translate.moztw.org) - Localization system related stuff 
* [wiki.moztw.org](https://github.com/moztw/wiki.moztw.org) - Wiki which is not in active usage
* [photos.moztw.org](https://github.com/moztw/photos.moztw.org) -  Picture Gallery of MozTW prior to Firefox 1.5 times.
